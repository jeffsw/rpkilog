lede

lede

The intro copy block at the top of rpkilog.com. Four short paragraphs. State → ask → credit → ask → thanks. Voice is sysadmin-direct.

rpkilog.com is available for use, but please consider it a work-in-progress! The source is available on GitHub: jeffsw/rpkilog.

Thanks to Job Snijders for making his VRP cache snapshots available.

This service runs on AWS. If you work there and can help get this project sponsored, please reach out by email.

Thanks!

fieldset

fieldset

The signature container — native <fieldset> + <legend>. The legend overlapping the border is iconic; do not fake it with divs.

rendered
RPKI History Search 
<fieldset class="rk-fieldset">
  <legend>RPKI History Search</legend>
  <div class="rk-form-row">
    <label class="rk-label">Prefix</label>
    <input class="rk-input" value="8.8.8.0/24">
    <button class="rk-btn" type="submit">Search</button>
  </div>
</fieldset>

input

input

Text input. Black bg, 1px green border, amber on focus. Use style="width:Npx" inline for per-field widths (matches existing rpkilog layout).

rendered · default / focused / disabled
<input class="rk-input" type="text" value="8.8.8.0/24">

button

button
